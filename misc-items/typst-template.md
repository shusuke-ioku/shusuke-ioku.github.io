### LaTeX-Like Typst Template

`TeXst` is a Typst package for clean LaTeX-like academic paper formatting.  
Use it for fast paper setup with academic defaults.

[Open Repository](https://github.com/shusuke-ioku/texst)

<span class="item-updated">Last Updated: 2026-02-12</span>
:::

::: {.item-more}
#### Installation (Typst Universe)

```bash
#import "@preview/texst:0.1.0": paper
```

#### Initialize A Template Project

```bash
typst init @preview/texst:0.1.0
```

#### Local Repository Checkout (Alternative)

```bash
#import "./src/lib.typ": paper
```
:::
